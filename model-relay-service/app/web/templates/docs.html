📖 API 对接文档

Base URL: {{ base_url }}（本地开发） / http://{{ local_ip }}:8000（局域网）
认证方式

本项目作为中转服务，客户端不需要额外认证。但请求格式需与配置的中转站服务类型匹配。

客户端直接使用 OpenAI 兼容的 SDK 设置 base_url 为本服务地址即可。

获取可用模型列表
curl {{ base_url }}/v1/models \
  -H "Authorization: Bearer sk-placeholder"

返回所有启用的模型（使用别名）。

OpenAI Chat Completions 格式

兼容 OpenAI SDK：

curl {{ base_url }}/v1/chat/completions \
  -H "Authorization: Bearer sk-placeholder" \
  -H "Content-Type: application/json" \
  -d '{
    "model": "your-model-alias",
    "messages": [{"role": "user", "content": "你好"}],
    "stream": false
  }'

Python 示例：

from openai import OpenAI

client = OpenAI(
    api_key="sk-placeholder",
    base_url="{{ base_url }}"
)
response = client.chat.completions.create(
    model="your-model-alias",
    messages=[{"role": "user", "content": "你好"}]
)
print(response.choices[0].message.content)
OpenAI Responses API
curl {{ base_url }}/v1/responses \
  -H "Authorization: Bearer sk-placeholder" \
  -H "Content-Type: application/json" \
  -d '{
    "model": "your-model-alias",
    "input": "你好，你是谁？"
  }'
Anthropic Messages API
curl {{ base_url }}/v1/messages \
  -H "x-api-key: sk-placeholder" \
  -H "content-type: application/json" \
  -H "anthropic-version: 2023-06-01" \
  -d '{
    "model": "your-model-alias",
    "max_tokens": 1024,
    "messages": [{"role": "user", "content": "你好，你是谁？"}]
  }'
Gemini GenerateContent API
curl "{{ base_url }}/v1beta/models/your-model-alias:generateContent" \
  -H "Content-Type: application/json" \
  -d '{
    "contents": [
      {"parts": [{"text": "你好，你是谁？"}]}
    ]
  }'
SDK 集成（通用）
Python (OpenAI SDK)
from openai import OpenAI

client = OpenAI(
    api_key="sk-placeholder",
    base_url="{{ base_url }}"
)

# 列出可用模型
models = client.models.list()
for m in models:
    print(m.id)

# 对话补全
response = client.chat.completions.create(
    model="deepseek-v4-flash",  # 使用别名
    messages=[{"role": "user", "content": "你好"}],
    stream=True
)
for chunk in response:
    print(chunk.choices[0].delta.content or "", end="")
Node.js (OpenAI SDK)
import OpenAI from 'openai';

const client = new OpenAI({
    apiKey: 'sk-placeholder',
    baseURL: '{{ base_url }}'
});

const response = await client.chat.completions.create({
    model: 'deepseek-v4-flash',
    messages: [{role: 'user', content: '你好'}],
    stream: true
});
for await (const chunk of response) {
    process.stdout.write(chunk.choices[0]?.delta?.content || '');
}
